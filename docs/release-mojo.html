paprika:release

Full name:

io.github.atos-digital-id:paprika-maven-plugin:0.1.0:release

Description:

Paprika release Mojo. Can be helpful to releasing modules: it displays or execute the Git commands for tagging the currently modified modules.

Attributes:

  • Requires a Maven project to be executed.
  • Executes as an aggregator plugin.
  • Executes by direct invocation only.

Optional Parameters

Name Type Since Description
<exec> boolean - Execute proposed commands.
Default value is: false.
User property is: exec.
<incrementPart> String - Part of version to increment. Can be MAJOR, MINOR or PATCH.
Default value is: MINOR.
User property is: increment.
<outputFile> File - Write the proposed commands in a file.
User property is: outputFile.
<releaseModules> boolean - Seek for sub-module to also release.
Default value is: true.
User property is: releaseModules.
